Import Master Data

Upload each CSV in order - lecturers, then courses, then students, then course registrations. Later steps look up rows created by earlier ones, so importing out of order will skip rows rather than fail outright. Re-uploading any file is safe - existing rows are updated, not duplicated.

1. Lecturers

Columns: staff_no, full_name, email

2. Courses

Columns: course_code, title, lecturer_staff_no (lecturers must already be imported)

3. Students

Columns: matric_no, full_name, department, level, email, phone, faculty, rfid_card_uid, photo_path (all but matric_no/full_name/department/level may be blank)

4. Course Registrations

Columns: matric_no, course_code (students and courses must already be imported)
